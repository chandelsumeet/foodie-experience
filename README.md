## How to run this project

Clone this project

```sh
git clone https://github.com/chandelsumeet/foodie-experience.git
```

Install dependencies

```sh
npm i
```

Run project locally

```sh
npm run dev
```

On your browser go to

```sh
 http://localhost:5173/
```

## How to build the project

```sh
npm run build
```
